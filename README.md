### Shortcuts

Adds configurable shortcut tiles to the degoog home page. Shortcuts are stored per-browser in localStorage and can be added, edited, or removed directly from the home page.

Features:
- **Auto-discovery**: Automatically fetches the site name and icon when you enter a URL.
- **Icon Picker**: Click the icon preview to choose from all icons available on the website (verified server-side).
- **Customization**: Set custom names, colors, and icon sizes for each shortcut.
- **Private Network Support**: Successfully grabs icons from self-hosted services on your local network (e.g., Jellyfin, Pi-hole) by fetching them server-side.
- **Per-browser storage**: Your shortcuts are unique to each browser/device you use.

### Prowlarr

Search your Prowlarr indexers directly from degoog via `!prowlarr`. Requires a running Prowlarr instance and API key. Supports custom sorting and Newznab category filtering.
